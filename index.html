Política de Privacidad — Pickly

Última actualización: 14 de noviembre de 2025

La presente Política de Privacidad describe de forma detallada cómo la aplicación Pickly (“la Aplicación”) gestiona la información del usuario durante su uso. Pickly ha sido desarrollada con un principio fundamental: no recopilar datos personales ni almacenar imágenes de los usuarios de forma permanente.

Pickly permite analizar entre 2 y 5 fotografías mediante un modelo de inteligencia artificial para determinar cuál es la más adecuada para su publicación en redes sociales. Toda la información procesada se maneja de manera temporal, segura y mínima.

1. INFORMACIÓN QUE RECOPILAMOS

1.1. Datos personales

Pickly no recopila, almacena ni procesa ningún dato personal del usuario. No existe registro, creación de cuentas ni inicio de sesión. La Aplicación no solicita nombre, correo electrónico, número de teléfono ni ningún tipo de información identificable.

1.2. Imágenes

El análisis de las fotografías funciona de la siguiente manera:

  • El usuario selecciona entre 2 y 5 imágenes desde su galería.
  • Las imágenes se envían temporalmente al servidor de Pickly para procesarlas mediante un modelo de IA.
  • Una vez generado el resultado (puntuación y recomendación), este se devuelve al dispositivo del usuario.
  • Las imágenes se eliminan automáticamente e inmediatamente del servidor al finalizar el proceso. No se almacenan, registran ni conservan copias.

La Aplicación tampoco conserva las imágenes en el dispositivo más allá del análisis activo. No se construyen historiales ni colecciones de fotografías del usuario.

1.3. Permisos del dispositivo

Para funcionar correctamente, Pickly puede solicitar:

  • Acceso completo o limitado a la galería de fotos, únicamente para permitir que el usuario seleccione imágenes.
  • Permiso para enviar notificaciones, si el usuario decide activarlas.

Estos permisos pueden ser modificados o revocados en cualquier momento desde los ajustes del dispositivo.

1.4. Datos técnicos o de uso

Pickly no recopila:

  • Identificadores únicos del dispositivo
  • Direcciones IP
  • Estadísticas de uso o actividad
  • Datos de rendimiento
  • Información sobre hardware o software
  • Cualquier tipo de dato que permita analizar el comportamiento del usuario

Tampoco utiliza cookies, herramientas de analítica ni sistemas de seguimiento.

1.5. Información de fallos

Pickly no integra plataformas de crash analytics (como Firebase Crashlytics, Sentry o similares). Por tanto, no se registran errores, bloqueos, reportes automáticos ni información técnica del dispositivo.

2. FINALIDAD DEL TRATAMIENTO

La única finalidad del tratamiento de datos es:

  • Procesar temporalmente las fotografías seleccionadas para generar una puntuación mediante IA.
  • Mostrar el resultado al usuario.

Las imágenes no se utilizan para:

  • Entrenar modelos de inteligencia artificial
  • Mejorar algoritmos
  • Realizar análisis estadísticos
  • Generar perfiles del usuario
  • Actividades comerciales o publicitarias

3. CUSTODIA, CESIÓN Y COMUNICACIÓN DE DATOS

Pickly no cede, comparte ni vende información ni imágenes del usuario a terceros.

Las fotografías únicamente se transmiten al servidor de Pickly con carácter temporal y exclusivamente para procesarlas. No se envían a proveedores externos, plataformas de IA de terceros ni servicios ajenos al proyecto.

4. PROCESAMIENTO DE IMÁGENES

4.1. Procesamiento en servidor

El análisis de las imágenes se realiza en un servidor seguro administrado por los desarrolladores de Pickly. El flujo de tratamiento es el siguiente:

  1. El usuario selecciona las fotografías.
  2. La Aplicación envía las imágenes mediante una conexión cifrada (HTTPS/SSL).
  3. El servidor procesa las imágenes mediante IA y genera una puntuación.
  4. El resultado se envía nuevamente al usuario.
  5. Las imágenes se eliminan de manera definitiva del servidor.

4.2. Retención

Tiempo de conservación de imágenes: 0 minutos.
Una vez devuelto el resultado:

  • Las fotografías se borran inmediatamente.
  • No se almacenan en bases de datos.
  • No existe copia de seguridad ni histórico.

4.3. Uso prohibido

Las imágenes del usuario no se emplean para:

  • Entrenamiento o reentrenamiento de modelos de IA
  • Construcción de bases de datos
  • Estudios estadísticos
  • Investigación interna o externa
  • Publicidad o marketing

5. SEGURIDAD DE LA INFORMACIÓN

Pickly aplica medidas técnicas y organizativas adecuadas para proteger la información durante el procesamiento:

  • Transmisión cifrada mediante HTTPS/SSL
  • Eliminación automática de imágenes tras el análisis
  • Servidor seguro y protegido
  • Ausencia de almacenamiento permanente

Aun así, el usuario debe tener en cuenta que ninguna transmisión de datos por Internet es completamente infalible.

6. DERECHOS DEL USUARIO

El usuario conserva pleno control sobre sus imágenes y su privacidad. Puede:

  • Revocar permisos de acceso a la galería o a las notificaciones desde los ajustes del dispositivo en cualquier momento.
  • Eliminar la Aplicación cuando lo desee.
  • Usar la Aplicación sin proporcionar datos personales.

Dado que no recopilamos datos personales, no existen datos que puedan ser solicitados, modificados o eliminados en virtud del RGPD.

7. SERVICIOS Y ENLACES DE TERCEROS

Pickly no incorpora:

  • Enlaces externos
  • Herramientas de analítica
  • Servicios de monetización
  • Plataformas de terceros como Firebase, Mixpanel, Superwall o RevenueCat

El único servicio implicado es el servidor propio encargado del procesamiento temporal de imágenes.

8. CAMBIOS EN ESTA POLÍTICA

Podemos modificar esta Política de Privacidad en cualquier momento para adaptarla a cambios legales, técnicos o funcionales. La fecha de actualización indica la vigencia de la versión actual.
