Política de Privacidad — Pickly

Última actualización: 14 de noviembre de 2025

La presente Política de Privacidad describe cómo se tratan los datos cuando utiliza la aplicación móvil Pickly (“la Aplicación”), desarrollada y operada por sus creadores (“nosotros”).

1. Información que recopilamos

1.1. Datos personales

Pickly no recopila ni almacena ningún dato personal del usuario. No existe registro ni inicio de sesión.

1.2. Imágenes

El usuario selecciona entre 2 y 5 fotografías desde su galería. Estas imágenes se envían temporalmente a nuestro servidor únicamente para generar una puntuación mediante IA. Una vez enviado el resultado, las imágenes se eliminan automáticamente del servidor y no se conservan copias.

1.3. Permisos

La Aplicación puede solicitar acceso limitado o completo a la galería de fotos, y permiso para notificaciones. El usuario puede revocar estos permisos desde los ajustes del dispositivo.

1.4. Datos técnicos y de uso

Pickly no recopila estadísticas, identificadores del dispositivo, métricas ni utiliza herramientas de analítica.

1.5. Información de fallos

Pickly no usa herramientas de crash analytics y no registra datos sobre fallos.

2. Finalidad del tratamiento

La única finalidad es procesar las fotografías seleccionadas y generar una puntuación. No se usan para entrenar modelos, mejorar servicios ni realizar análisis estadísticos.

3. Cesión de datos

No se comparten ni venden imágenes a terceros. Las fotos solo se envían a nuestro servidor propio para su análisis temporal.

4. Procesamiento de imágenes

El procesamiento se realiza en un servidor seguro y las imágenes se eliminan tras completar la operación.

5. Seguridad

La comunicación con el servidor está cifrada (HTTPS/SSL). Las imágenes se eliminan inmediatamente después del análisis.

6. Derechos del usuario

El usuario puede revocar permisos de acceso a fotos o notificaciones y eliminar la Aplicación en cualquier momento. No existe almacenamiento de datos personales, por lo que no aplican solicitudes de acceso o rectificación.

7. Servicios de terceros

Pickly no utiliza plataformas externas como Firebase, Mixpanel, RevenueCat o similares.

8. Cambios

Podemos actualizar esta Política de Privacidad en cualquier momento. La fecha indica la versión vigente.
