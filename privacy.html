Política de Privacidad — Pickly Última actualización: 14 de noviembre de 2025 La presente Política de Privacidad describe el tratamiento de datos llevado a cabo cuando utiliza la aplicación móvil Pickly (“la Aplicación”), desarrollada y operada por sus creadores (“nosotros”). Pickly está diseñada para ofrecer un análisis de fotografías mediante un sistema de puntuación impulsado por un modelo de inteligencia artificial. Nos comprometemos a proteger su privacidad y garantizar un tratamiento seguro y responsable de los datos. 1. INFORMACIÓN QUE RECOPILAMOS 1.1. Datos personales Pickly no recopila, almacena ni procesa datos personales del usuario. No existe registro, inicio de sesión ni creación de cuentas. 1.2. Imágenes Para realizar el análisis y puntuación de fotografías, el usuario selecciona entre 2 y 5 imágenes desde su galería. El funcionamiento es el siguiente: Las imágenes seleccionadas se envían temporalmente a nuestro servidor para procesarlas mediante un modelo de IA. Tras generar la puntuación y devolver el resultado al usuario: Las imágenes se eliminan automáticamente e inmediatamente del servidor. No se almacenan, registran ni conservan copias. La Aplicación tampoco conserva las imágenes más allá del proceso activo de análisis. 1.3. Permisos del dispositivo La Aplicación puede solicitar: Acceso completo o limitado a la galería de fotos, exclusivamente para permitir la selección de imágenes. Permiso de notificaciones, si el usuario desea habilitarlas. Estos permisos pueden modificarse o revocarse en cualquier momento desde los ajustes del dispositivo. 1.4. Datos técnicos o de uso Pickly no recopila datos técnicos, identificadores del dispositivo, estadísticas de uso, historial de actividad, ni utiliza herramientas de analítica o seguimiento. 1.5. Información de fallos Pickly no integra herramientas de análisis de errores o crash analytics, por lo que no se registran datos sobre fallos. 2. FINALIDAD DEL TRATAMIENTO La única finalidad del tratamiento de datos es: Procesar las fotografías seleccionadas por el usuario mediante un modelo de inteligencia artificial alojado en nuestro servidor. Devolver al usuario una puntuación y recomendación basada en ese procesamiento. No se da ningún otro uso a las imágenes ni se emplean para entrenar modelos, mejorar servicios o realizar análisis estadísticos. No se genera ningún tipo de perfil del usuario. 3. CUSTODIA, CESIÓN Y COMUNICACIÓN DE DATOS No cedemos, compartimos ni vendemos imágenes ni datos personales a terceros. Las fotografías se envían únicamente a nuestro propio servidor, estrictamente para posibilitar la funcionalidad principal de la Aplicación. No se transmiten imágenes a terceros proveedores, plataformas de IA externas ni servicios ajenos al proyecto. 4. PROCESAMIENTO DE IMÁGENES 4.1. Procesamiento en servidor El análisis de las imágenes se realiza en un servidor controlado por nosotros. Se aplica el siguiente flujo: El usuario selecciona las fotografías. La Aplicación las envía de forma cifrada al servidor. El servidor procesa las imágenes y genera la puntuación. Se devuelve el resultado al usuario. Las imágenes son eliminadas definitivamente del servidor al finalizar la operación. 4.2. Retención Tiempo de conservación de imágenes: 0 minutos. Una vez devuelto el resultado, las fotografías se eliminan de forma irreversible. 4.3. Uso prohibido Las fotografías no se utilizan para: Entrenar modelos de IA Construir bases de datos Realizar análisis estadísticos Mejorar algoritmos Actividades publicitarias o comerciales 5. SEGURIDAD DE LA INFORMACIÓN Aplicamos medidas técnicas y organizativas para proteger la confidencialidad e integridad de las imágenes procesadas. Entre ellas: Transmisión cifrada (HTTPS/SSL) Eliminación automática tras el análisis Servidores seguros y protegidos Ausencia total de almacenamiento permanente Sin embargo, el usuario debe tener en cuenta que ninguna transmisión de datos por Internet es completamente infalible, aunque implementamos las medidas necesarias para minimizar riesgos. 6. DERECHOS DEL USUARIO El usuario tiene derecho a: Revocar permisos de acceso a fotos o notificaciones en cualquier momento. Eliminar la Aplicación cuando lo desee. Controlar completamente sus propias imágenes, ya que no conservamos ninguna copia. Dado que no almacenamos datos personales, no existen datos susceptibles de acceso, rectificación o supresión en virtud del RGPD. 7. SERVICIOS Y ENLACES DE TERCEROS Pickly no incorpora enlaces externos ni integra servicios de terceros que procesen datos del usuario, salvo el propio servidor de análisis operado por nosotros. No se utilizan servicios como Firebase, Mixpanel, RevenueCat, Superwall ni similares. 8. CAMBIOS EN ESTA POLÍTICA Podemos actualizar esta Política de Privacidad para adaptarnos a cambios legales, técnicos o funcionales. La fecha de última actualización indica la vigencia de la versión actual. 9. CONTACTO Para dudas o consultas sobre esta Política de Privacidad, puede contactarnos en: pickly_es@icloud.com
