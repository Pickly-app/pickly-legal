Política de Privacidad — Pickly

Última actualización: 14 de noviembre de 2025

La presente Política de Privacidad describe el tratamiento de datos realizado cuando utiliza la aplicación móvil Pickly (“la Aplicación”). Pickly procesa temporalmente imágenes para analizarlas mediante un modelo de inteligencia artificial y devolver una puntuación al usuario.

1. Información que recopilamos

1.1. Datos personales

Pickly no recopila, almacena ni procesa datos personales del usuario. No existe registro, inicio de sesión ni creación de cuentas.

1.2. Imágenes

El usuario selecciona entre 2 y 5 imágenes desde su galería. Estas se envían temporalmente al servidor para ser procesadas por un modelo de IA. Tras devolver el resultado, las imágenes se eliminan automáticamente y no se almacenan.

1.3. Permisos del dispositivo

La Aplicación puede solicitar acceso a la galería de fotos y permiso de notificaciones. El usuario puede modificarlos o revocarlos en cualquier momento.

1.4. Datos técnicos o de uso

No recopilamos datos del dispositivo, identificadores, estadísticas de uso ni empleamos herramientas de analítica.

1.5. Información de fallos

Pickly no incorpora herramientas de crash analytics, por lo que no se registran fallos.

2. Finalidad del tratamiento

La única finalidad es procesar temporalmente las imágenes mediante IA para devolver una puntuación y recomendación. No se utilizan para entrenar modelos ni realizar análisis estadísticos.

3. Cesión y comunicación de datos

No cedemos, compartimos ni vendemos imágenes ni datos personales a terceros. Las imágenes solo se envían a nuestro propio servidor para su procesamiento.

4. Procesamiento de imágenes

4.1. Procesamiento en servidor

El análisis se realiza en un servidor propio. La transmisión es cifrada y las imágenes se eliminan al finalizar.

4.2. Retención

Las imágenes se conservan durante 0 minutos. Se eliminan inmediatamente tras la operación.

4.3. Uso prohibido

Las imágenes no se utilizan para entrenar modelos, bases de datos, análisis estadísticos ni fines comerciales.

5. Seguridad

Aplicamos medidas de seguridad como transmisión cifrada, eliminación automática, servidores seguros y ausencia de almacenamiento permanente.

6. Derechos del usuario

El usuario puede revocar permisos, eliminar la aplicación o controlar sus imágenes. No almacenamos datos personales, por lo que no aplican derechos de acceso, rectificación o supresión.

7. Servicios de terceros

Pickly no integra servicios de terceros como Firebase, Mixpanel, RevenueCat o similares.

8. Cambios en la política

Podremos actualizar esta Política de Privacidad en caso de cambios legales, técnicos o funcionales.
